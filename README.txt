Welcome to the "Motube" WordPress Plugin!

This Plugin provides a fancy 3D-video-navigation platform for the iPhone.


Installation Instructions:

1. Download the Plugin and load it up to the wp-content/plugins/ -folder OR install it via the built-in plugin installer

2. Copy "motube-for-theme.php" to wp-content/themes/YOURTHEME/

3. Activate the Plugin

4. Create a new page in the admin interface with the page-template "MoTube"

5. Add and categorize as many videos as you want (upload manually or via media library)

6. Enjoy!!!
